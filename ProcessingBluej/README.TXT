## BluejProcessingWrapper
Dieses Programm dient als wrapper bibiothek für die processing java bibiothek und PDE in bluej.
Da es keine rendering bibiothek innerhalb bluej gab wurde diese wrapper bibiothek als übergangslösung entwickelt.
Die processing bibiothek bietet keine mölichkeit die aus bluej gewöhnten UML diagramme und aufrufsdialoge anzuzeigen.
Processing bietet aber die möglichkeit die rendering bibiothek in einem gewöhnlichen java kontext zu verwenden.
